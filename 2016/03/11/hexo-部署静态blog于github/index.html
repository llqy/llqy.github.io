hexo 部署静态blog于github

配置环境

OS: win7 64位

安装Node & git:

到Nodejs.org 官网下载安装最新版.

到git-scm.com官网下载安装最新版.

开始行动

注册github

到github.com注册帐号,新建一个username.github.io的repository.
username替换成你的用户名!!必须如此

添加SSH key

打开Git Bash(安装git后在开始菜单里有的).

1.首先设置你的用户名密码：

llqy5251@qq.com 改成你的邮箱,此邮箱只是作为之后上传者的身份邮箱,llqy改成你的名字.

2.生成密钥：

ssh-keygen -t rsa -C “llqy5251@qq.com”
同上,llqy5251@qq.com改成你的邮箱,其实改成其他的也无所谓,不影响后面.

其中会提示你输入保存key的地址,不改直接回车的话会在默认的c:\users\username,username是你的系统的用户名.

用记事本打开生成的ssh.pub,复制里面的
